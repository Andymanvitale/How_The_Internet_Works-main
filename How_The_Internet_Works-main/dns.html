DNS(Domain Naming System)

DNS is the standard form for naming websites and has been adopted internationally. It relates an IP address to a more easy to remember website name. It wouldn't be practical to remember the number combination of your favorite website, but using DNS like a phone book, you can type in what you are looking for and DNS will search for it and connect you to what you want to see.

Unfortunately, having a public network of available addresses and websites calls for trouble to start. Someone can spoof the IP address of the website you are trying to find and you can become susceptible to cyber attacks or having personal information stolen. It's hard to tell if you're really on the website you believe you are on, but having a secure connection can verify whether or not you are on the site you are on.

How every website is tracked

The internet is a large collection of many websites and data, so how can each site have it's own unique name and address. This is referred to in the IP Addresses page, but an IP address is assigned to it, and that IP address is connected to the domain name it holds. There are regulating organizations which oversee this process, such as ICANN. Furthermore, looking at the breakdown of how a domain is structured above, the root can be .org, .com, or .net, among many others. Any of these three top level domains can contain the same domain name. Having these structures allows for the interconnectivity at nearly instant speeds to occur. Any additional translation of data or packets of data being lost due to incompatible domains seems unacceptable.
